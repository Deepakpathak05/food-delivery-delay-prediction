# Food Delivery Operations Analysis and Delay Prediction

## Project Overview
This project analyzes food delivery operations using Python and develops a machine learning model to predict delivery delays.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Business Insights
- Data Visualization
- Machine Learning

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Key Insights
- Delivery distance and traffic congestion are major contributors to delivery delays.
- Weather conditions moderately impact delivery efficiency.
- Logistic Regression demonstrated better recall for delayed deliveries under class imbalance conditions.

## Machine Learning Models
- Random Forest Classifier
- Logistic Regression
